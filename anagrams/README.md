# anagrams -- Displays arrays with all anagrams

Problem is given a list or array like this:

```
let list = vec![String::from("ear"),
                String::from("era"),
                String::from("mug"),
                String::from("gum"),
                String::from("are"),
                String::from("ear"),
                String::from("sit")];
```

Solution is in Rust:

- anagrams.rs - Solution in Rust. Compile with: rustc anagrams.rs
              - Run with: ./anagrams
